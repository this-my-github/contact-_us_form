## Начало работы

Чтобы запустить проект, используйте следующие команды:

### Сборка проекта

```bash
npm run build
```
Создает сборку вашего проекта.

### Запуск сервера

```bash
npm run server
```
Запускает сервер для обработки запросов

### Просмотр проекта

Откройте файл src/index.html в браузере
